# Customer Segmentation Using Clustering

This repository contains a **Machine Learning final project** focused on **customer segmentation using clustering techniques**. The goal of this project is to analyze customer data from a marketing campaign and group customers based on similar characteristics and purchasing behavior.

Customer segmentation helps companies better understand their customers and develop **more targeted and effective marketing strategies**.

## Project Overview

In this project, we analyze a marketing campaign dataset that contains information about customer demographics, income, purchasing behavior, and responses to previous campaigns. By applying clustering methods, customers are grouped into several segments based on similarities in their data.

The results of this analysis can help businesses identify different customer groups and tailor marketing strategies for each segment.

## Dataset

The dataset used in this project is **Marketing Campaign Dataset**, which contains information such as:

* Age and demographic information
* Education level
* Marital status
* Income
* Product purchasing behavior
* Customer responses to marketing campaigns

## Methodology

The analysis process includes several stages:

1. **Exploratory Data Analysis (EDA)**
   Understanding the dataset using descriptive statistics and visualization.

2. **Data Preprocessing**

   * Handling missing values
   * Data cleaning
   * Feature selection

3. **Clustering Model**
   Applying clustering algorithms to group customers with similar characteristics.

4. **Cluster Analysis**
   Interpreting the characteristics of each customer segment.
